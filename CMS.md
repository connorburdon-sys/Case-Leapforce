In dit document heb ik per text veld over de CMS benodigheden nagedacht en heb ik deze genoteerd.

label:
    class naam = label
    standaard waarde = label
    maximaal aantal characters = 50 
    lijnen bevatten aanpasbare text (in index.html) = 15
    Het label is in deze sectie een non verplicht onderdeel waarin u text in een kleiner formaat kan neerzetten. Denk bijvoorbeeld aan de naam van uw bedrijf of het onderwerp van de sectie.

Titel section:
    class namen = heading1 , heading1Highlight
    standaard waarde = Mandatory title about the call-to-action here
    maximaal aantal characters = 45
    lijnen bevatten aanpasbare text (in index.html) = 18 tm 20
    De titel is een verplicht onderdeel van de call-to-action waar je een optionele highlight bij kan plaatsen, is deze niet nodig kunt u lijn 19 in de file index.html aanpassen.

body text:
    class naam = textLarge
    standaard waarde = Optional body copy goes here.
    maximaal aantal characters = 70 (meer dan dit heeft de <br> nodig zodat de tekst niet te breed wordt)
    lijnen bevatten aanpasbare text (in index.html) = 23
    De body text is een optioneel onderdeel van deze sectie die wat meer uitleg over het onderwerp geeft als dat nodig is.

button:
    class naam = tertiaryCTA
    standaard waarde = Optional button here (met een pijl icoon)
    maximaal aantal characters = 70
    lijnen bevatten aanpasbare text (in index.html) = 25
    Deze knop is een optionele toevoeging voor als u de lezer bijvoorbeeld door wilt sturen naar informatie over het onderwerp of bronnen van de informatie die uw weer geeft.
    als u de knop niet nodig heeft kunt u lijn 25 tm 32 verwijderen.

achtergrond foto cards:
    als u andere foto's wilt hebben als achtergrond bij de kaarten kunt u contact opnemen en dan passen we ze aan.

cards titels:
    class naam = heading4 
    standaard waarde = Title
    maximaal aantal characters = 50 (voordat de tekst 2 lijnen bevat)
    lijnen bevatten aanpasbare text (in index.html) = 40 en 58
    Dit is een verplicht veld die u kunt invullen voor een individueel verhaal in de cta.

supporting text:
    class naam: cardText 
    standaard waarde: Supporting text about the call-to-action goes here.
    maximaal aantal characters = 80
    lijnen bevatten aanpasbare text (in index.html) = 43 en 61
    Dit is een verplicht veld om de story die u promoot wat beter uit te leggen in een korte krachtige zin.
